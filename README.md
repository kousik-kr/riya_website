# Professor Personal Website

A modular, responsive personal website template for a senior professor.

## Features
- Professional, minimal color system using design tokens.
- Responsive layout for desktop/tablet/mobile.
- Floating card animations for subtle visual depth.
- Separated components, styles, and design token files.
- Accessible semantic structure and skip link.

## Project Structure
- `index.html` — application shell.
- `scripts/main.js` — composition root.
- `scripts/components/*.js` — section-level UI components.
- `styles/design-tokens.css` — color, spacing, and visual tokens.
- `styles/main.css` — base and component styling.

## Run Locally
Open `index.html` directly, or run a static server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
